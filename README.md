### Карта репозитория

```
├── ab_test_platform
│ ├── coverage_html
│ │ ├── index.html -- процент покрытия тестами (полный отчет по покрытию)
│ ├── Dockerfile
│ ├── requirements.txt
│ ├── scripts - вспомогательные скрипты
│ └── src
│     ├── api
│     │ ├── deps.py - DI через Depends фастапи
│     │ └── routes - роуты API
│     ├── application -- бизнес-логика, взаимодействует через репозитории с базами данных
│     │ ├── decisions_service.py -- КРИТИЧНЫЙ ПОТОК - принятие решений
│     │ ├── events_sevice.py -- КРИТИЧНЫЙ ПОТОК - получение и обработка batch а событий
│     │ ├── guardrail_service.py -- КРИТИЧНЫЙ ПОТОК - проверка guardrails
│     │ ├── reports_service.py -- КРИТИЧНЫЙ ПОТОК - точка экспорта метрик (отчеты)
│     │ └── worker.py -- проверка раз в сколько-то секунд guardrails и перестройка MW
│     ├── core -- скрипты инициализации / сброса бд, обработка переменных окружения
│     ├── domain -- интерфейсы инфраструктурного слоя и описание известных ошибок (exceptions) которые могут райзиться откуда угодно
│     ├── infra -- инфраструктурный слой
│     ├── main.py -- запуск проекта
│     ├── models -- основные сущности сервиса, они же описание схемы базы данных для ORM sqlmodel
│     ├── schemas -- схемы ответов/запросов в апи и взаимодействия с данными в application слое
├── demo -- данные для демо и ручного тестирования, а также документация продукта.
├── docker -- init данные докера
├── docker-compose.yml
├── external_system_emulator -- система-эмулятор (отдельный FastAPI сервис)
├── README.md
└── tests -- тесты
```
